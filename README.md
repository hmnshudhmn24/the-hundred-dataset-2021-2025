# The Hundred Dataset (2021–2025)

A detailed ball-by-ball dataset covering **322 matches** from The Hundred played between **2021 and 2025**.
This dataset includes delivery-level events, runs, wickets, extras, player actions, and complete match metadata—ideal for cricket analytics, machine learning, visualization, and research.

## Features
- 322 matches from The Hundred
- Ball-by-ball delivery data
- Runs, wickets, extras, dismissals
- Batting, bowling, and fielding details
- Match metadata (teams, venue, date, toss, result)

## Format
Available in **JSON** and **YAML** formats.
